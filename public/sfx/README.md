# Sound Effects Library

## Quick Start

**TLDR:** Download 4 sound effects from Freesound or Mixkit, save to this directory.

📋 **See `RECOMMENDED-SOUNDS.md` for specific file recommendations**
📖 **See `DOWNLOAD-GUIDE.md` for detailed download instructions**

---

## Target Files

You need to download and place these 4 files in this directory:

| File | Purpose | Duration | Use Case |
|------|---------|----------|----------|
| `whoosh-in.mp3` | Scene transitions | 0.4-1s | Slide-ins, scene changes |
| `impact-soft.mp3` | UI reveals | 0.5-1s | Card appears, UI entrance |
| `impact-hard.mp3` | Stat reveals | 1-1.5s | Number slam, emphasis |
| `ui-click.mp3` | Micro-interactions | 0.2-0.3s | Button hover, clicks |

## Requirements

All sound effects must be:
- ✓ High quality (44.1kHz or better)
- ✓ Short duration (0.3-1.5s)
- ✓ Royalty-free for commercial use
- ✓ MP3 or WAV format (WAV can be converted)
